# Job Tracker

A job application tracker built for international students — track where you've applied,
which resume version you used, and whether a role is realistic for your visa situation
(OPT, STEM OPT, H1B sponsorship), all in one place.

Built with React.

## 🚧 Status: In progress

Early development. The project is scaffolded and core features are being built next.

## The problem

Students apply to hundreds of jobs and lose track of what they sent where — and
international students have an extra layer: figuring out which roles even support
their visa status. This app aims to solve both.

## Features

**In progress**
- Project setup with React
- Add and track applications (company, role, date, status)

**Planned**
- Resume version tracking — see which resume was used for each application and which performs best
- Visa tags per job: OPT / STEM OPT friendly, sponsorship needed (Yes / No / Unknown)
- Dashboard: total applications, interviews, response rate
- Follow-up reminders
- Cross-platform access (web + mobile)

## Tech stack
React · (Firebase planned for auth, database, and resume storage)
